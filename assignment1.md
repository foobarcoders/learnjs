### एक array के सारे नंबर को double करो
example:
input `[4,33,25,15]`
output `[8,66,50,30]`

### एक object में student के marks लो और marks को जोड़ कर (total करके) उसी object me एक key `total` बना कर सवे करके print करो
example:
input 
```
let m = {
subj1: 64,
subj2: 74,
subj3: 84,
subj4: 68,
subj5: 60,
subj6: 62
};
```
output
```
{
subj1: 64,
subj2: 74,
subj3: 84,
subj4: 68,
subj5: 60,
subj6: 62,
total: 412
};
```
**hint** use Object.keys(marks) and use for loop
